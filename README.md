A library of additional parts for KiCad for use with Sakura-IT projects.

Currently includes:

* Motorola processors:
  * MC68000P
  * MC68000FN
  * MC68012RC
  * MC68020RC
  * MC68030FE
* Amiga components:
  * A1200 clockport
  * A1200 Gayle
  * A3000/A4000 local bus
* Compact Flash card connector
* FTDI FT245RL (USB-to-serial parallel bridge)
* Xilinx JTAG port
* Xilinx XC9536XL
